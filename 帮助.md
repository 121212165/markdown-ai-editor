# Markdown 编辑器帮助文档

## 简介

欢迎使用我们的 Markdown 编辑器！这是一个功能丰富、界面友好的 Markdown 文本编辑器，支持实时预览、语法高亮、快捷键操作等功能。

## 基本操作

### 编辑区域

左侧是 Markdown 文本编辑区域，您可以在这里编写 Markdown 格式的文本。编辑器支持以下功能：

- 实时语法高亮
- 行号显示
- 自动补全
- 括号匹配

### 预览区域

右侧是实时预览区域，您在编辑区域输入的内容会立即在此处渲染成 HTML 格式显示。

### 工具栏

顶部工具栏提供了常用的 Markdown 格式化按钮：

- **B**: 粗体
- *I*: 斜体
- #: 标题
- 🔗: 链接
- 🖼️: 图片
- ☰: 列表
- 💻: 代码块

### 文件操作

- 💾 保存: 将当前文档保存为 .md 文件
- 📂 打开: 打开本地的 Markdown 文件
- 👁️ 预览: 切换预览模式（仅显示预览区域）

## Markdown 语法指南

### 标题

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 文本样式

```
**粗体文本**
*斜体文本*
~~删除线文本~~
`行内代码`
```

### 列表

无序列表：
```
- 项目1
- 项目2
  - 子项目
```

有序列表：
```
1. 第一项
2. 第二项
3. 第三项
```

### 代码

行内代码：`console.log('Hello World')`

代码块：
```markdown
​```javascript
function hello() {
  console.log('Hello World');
}
​```
```

### 链接和图片

```
[链接文本](链接地址)
![图片描述](图片地址)
```

### 引用

```
> 这是一段引用文本
```

### 表格

```
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容1 | 内容2 | 内容3 |
```

### 分割线

```
---
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl + B | 粗体 |
| Ctrl + I | 斜体 |
| Ctrl + K | 插入链接 |
| Ctrl + Shift + I | 插入图片 |
| Ctrl + L | 插入列表 |
| Ctrl + Shift + C | 插入代码块 |
| Ctrl + P | 预览切换 |
| Ctrl + S | 保存文件 |
| Ctrl + O | 打开文件 |

## 常见问题

### 1. 如何保存我的文档？

点击工具栏中的保存按钮（💾）或使用快捷键 Ctrl + S。

### 2. 如何打开已有的 Markdown 文件？

点击工具栏中的打开按钮（📂）或使用快捷键 Ctrl + O。

### 3. 如何只查看预览效果？

点击工具栏中的预览按钮（👁️）或使用快捷键 Ctrl + P。

### 4. 支持哪些 Markdown 语法？

本编辑器支持 CommonMark 标准和 GitHub Flavored Markdown 扩展语法。
