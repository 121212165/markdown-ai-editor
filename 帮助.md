# Markdown 编辑器帮助文档

## 简介

这是一个基于浏览器的 Markdown 编辑器，支持实时预览和常用格式化操作。

## 界面

- **顶部工具栏**：格式化按钮和文件操作按钮
- **左侧**：Markdown 编辑区域
- **右侧**：实时预览区域
- **底部状态栏**：显示当前行号、列号、字数和预估阅读时间

## 工具栏按钮

- **B**：插入粗体
- **I**：插入斜体
- **#**：插入标题
- **链接**：插入链接
- **图片**：插入图片
- **列表**：插入无序列表
- **代码块**：插入代码块
- **预览**：切换预览模式（仅显示渲染结果）
- **保存**：下载当前内容为 .md 文件
- **打开**：打开本地 .md 文件

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl + B | 粗体 |
| Ctrl + I | 斜体 |
| Ctrl + K | 插入链接 |
| Ctrl + Shift + I | 插入图片 |
| Ctrl + L | 插入列表 |
| Ctrl + Shift + C | 插入代码块 |
| Ctrl + P | 切换预览模式 |
| Ctrl + S | 保存文件 |
| Ctrl + O | 打开文件 |

## Markdown 语法参考

### 标题

```
# 一级标题
## 二级标题
### 三级标题
```

### 文本样式

```
**粗体**
*斜体*
~~删除线~~
`行内代码`
```

### 列表

```
- 无序列表项
1. 有序列表项
```

### 链接和图片

```
[链接文本](URL)
![图片描述](URL)
```

### 引用

```
> 引用文本
```

### 代码块

````
```
代码内容
```
````

### 表格

```
| 列1 | 列2 |
|-----|-----|
| 内容 | 内容 |
```
