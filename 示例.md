# 欢迎使用 Markdown 编辑器

这是一个功能强大的 Markdown 编辑器，支持实时预览和多种实用功能。

## 主要功能

### 1. 文本格式化
- **粗体文本** - 使用两个星号包围文字
- *斜体文本* - 使用单个星号包围文字
- ~~删除线~~ - 使用两个波浪号包围文字
- `行内代码` - 使用反引号包围代码

### 2. 代码块
```javascript
// 这是一段 JavaScript 代码示例
function helloWorld() {
  console.log('Hello, world!');
}
```

### 3. 列表

无序列表：
- 项目 1
- 项目 2
- 项目 3

有序列表：
1. 第一项
2. 第二项
3. 第三项

### 4. 表格

| 姓名 | 年龄 | 城市 |
| ---- | ---- | ---- |
| 张三 | 25   | 北京 |
| 李四 | 30   | 上海 |
| 王五 | 28   | 广州 |

### 5. 引用

> 这是一个引用块。引用可以用来突出显示重要信息或引用他人的话语。

### 6. 链接和图片

[访问我们的网站](https://example.com)

![图片描述](https://example.com/image.jpg)

## 快捷键

- **Ctrl + B**: 粗体
- **Ctrl + I**: 斜体
- **Ctrl + K**: 插入链接
- **Ctrl + Shift + I**: 插入图片
- **Ctrl + L**: 插入列表
- **Ctrl + Shift + C**: 插入代码块
- **Ctrl + P**: 预览切换
- **Ctrl + S**: 保存文件
- **Ctrl + O**: 打开文件
